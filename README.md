# 🍔 Big Hammer Generator

A fun, interactive web application to randomly generate your next "Hammer" or "Big Hammer" sandwich! Designed with a sleek dark mode aesthetic, vibrant emojis, and satisfying slot-machine style animations.

You can try it live here: [Big Hammer Generator](https://big-hammer-generator.vercel.app/)

## ✨ Features

- **Two Sandwich sizes**: Choose between standard **Hammer** (1 meat) and **Big Hammer** (2 meats).
- **Random Generation**: Automatically picks your meat(s), 1 cheese, 3 raw veggies (crudités), and 1 sauce from a predefined tasty list.
- **Slot Machine Animations**: Enjoy a fun little slot machine spin animation when rolling your veggies.
- **Reroll**: Don't like your veggies? You can individually reroll the crudités for a better combo!
- **History Panel**: Keep track of your past pulls. Your history is saved locally in your browser. 
- **Custom Titles**: Name your favorite, weirdest, or tastiest sandwich creations straight from the history panel.
- **Easy Sharing**: Share your favorite sandwich configs with friends via native share (on mobile) or direct copy to clipboard formatting.

## 🛠️ Tech Stack

- **HTML5** & **CSS3** (Custom properties, animations, Flexbox/Grid)
- **Vanilla JavaScript** (ES6+, DOM manipulation, LocalStorage integration)
- Fully responsive, mobile-first design
- Font: [Outfit](https://fonts.google.com/specimen/Outfit) from Google Fonts

## 🚀 Getting Started

Since this is a vanilla frontend project with no build process, setting it up locally is as easy as slicing bread:

1. Clone the repository:
   ```bash
   git clone https://github.com/gabindeme/big-hammer-generator.git
   ```
2. Navigate to the project folder:
   ```bash
   cd big-hammer-generator
   ```
3. Open `index.html` in your favorite web browser or use a quick local server like Live Server (for VSCode).

## 💡 How it works

1. **Pick your type**: Select "Hammer" or "Big Hammer".
2. **Generate**: Click the big `🎲` button to spin the wheels.
3. **Save & Share**: Check the history (`📋` toggle) to name it, or hit the `📤` Share button to copy the recipe text to your clipboard.
